# 🖐️ Virtual Mouse Control using Computer Vision

![Python](https://img.shields.io/badge/python-3.11+-blue.svg)
![OpenCV](https://img.shields.io/badge/OpenCV-Library-green.svg)
![MediaPipe](https://img.shields.io/badge/MediaPipe-Google-orange.svg)

A cutting-edge Python application that allows you to control your computer mouse cursor using hand gestures. By leveraging **MediaPipe** for hand landmark detection and **PyAutoGUI** for system control, this project transforms your webcam into a powerful human-computer interface (HCI).

## ✨ Features
- 🚀 **Real-time Tracking:** High-speed hand landmark detection.
- 🖱️ **Gesture Control:** Move the mouse cursor by pointing your index finger.
- 🖥️ **Screen Mapping:** Automatically maps hand coordinates to your screen resolution.
- 🎨 **Visual Feedback:** Real-time drawing of hand connections on the video feed.

## 🛠️ Technologies Used
- **Python:** Core programming language.
- **OpenCV:** For image processing and video stream handling.
- **MediaPipe:** For advanced hand landmark detection.
- **PyAutoGUI:** To interface with the OS and control the mouse.
