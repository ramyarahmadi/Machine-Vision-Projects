# Machine-Vision-Projects 👁️
Welcome to my Machine Vision Projects Repository.

A curated collection of Machine Vision and Computer Vision projects, ranging from real-time hand gesture recognition to advanced object detection using OpenCV, MediaPipe, and Deep Learning.

This repository contains a collection of my experiments and projects in Computer Vision, focusing on real-time image processing, hand tracking, and intelligent visual systems using Python.
Overview

The goal of this repository is to explore practical applications of machine vision and build solutions to real-world problems using modern computer vision tools and libraries.

Projects here focus on:

    Real-time hand tracking and gesture recognition
    Image processing and visual feature extraction
    Human–computer interaction using vision systems
    Practical implementations using modern CV frameworks

Technologies Used

The projects in this repository mainly use:

    Python
    OpenCV
    MediaPipe
    NumPy
    Computer Vision Algorithms

Repository Structure

Each project is organized in its own folder and contains its own code and documentation.

Learning Focus

Rather than only implementing code, these projects focus on understanding the logic and mathematics behind computer vision techniques, including:

    Landmark-based hand tracking
    Relative geometric relationships between points
    Handling mirrored camera feeds
    Robust gesture detection under different orientations

Future Work

This repository will continue to grow with additional projects such as:

    Object detection systems
    Face landmark analysis
    Gesture-based control interfaces
    Deep learning–based vision models
